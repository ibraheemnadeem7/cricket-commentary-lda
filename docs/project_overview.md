# Project Overview

## Title
**What Is the Difference in Commentary for a Test Match and a T20 Match?  
A Topic Modeling Analysis Using LDA**

---

## Background

Cricket is the second most popular sport globally, with an estimated 2.5 billion fans. As the sport has evolved, so too have its formats â€” most notably the emergence of **Twenty20 (T20)** cricket alongside the traditional **Test format**.

While gameplay differences between these formats are well studied, **how commentary language changes across formats has received little formal analysis**. Anecdotally, long-time fans believe that Test cricket commentary is more technical and analytical, whereas T20 commentary prioritizes accessibility and entertainment.

This project addresses that gap using **Latent Dirichlet Allocation (LDA)** to quantitatively analyze and compare commentary language across formats.

---

## Research Question

> How does the language and thematic structure of cricket commentary differ between Test cricket and T20 cricket?

---

## Project Scope

- **Data**: English-language, ball-by-ball commentary transcripts
- **Formats analyzed**:
  - 23 Test matches
  - 15 T20 matches
- **Method**: Unsupervised topic modeling using LDA
- **Focus**: Linguistic themes, technical vocabulary, narrative structure

Out of scope:
- Sentiment analysis
- Audio/video analysis
- Player or match outcome prediction

---

## Contributions

- One of the first NLP-based comparative analyses of cricket commentary
- Large-scale corpus analysis (>850,000 total words)
- Empirical evidence supporting format-driven commentary differences
