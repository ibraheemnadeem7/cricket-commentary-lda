# Data Sources

## Commentary Transcripts

All commentary data was derived from **publicly available YouTube videos** of international cricket matches.

Only full-length matches were used to avoid bias.

---

## Video Sources

- Windies Cricket (YouTube)
- Pakistan Cricket (YouTube)
- England & Wales Cricket Board (YouTube)
- T Sports (YouTube)

---

## Transcript Extraction Tool

- youtubetotranscript.com
